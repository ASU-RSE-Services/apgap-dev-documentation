# APGAP Developer Documentation

The developer documentation site is build using [Zensical](https://zensical.org/docs). Please refer to their documentation for contributing. The site is deployed in [Read the Docs](https://about.readthedocs.com/).

## Local Development

To run this site locally, run `uv run zensical serve` in the root folder.

## Deployment

Pushes to the `main` branch of the GitHub repository will trigger a redeploy in [readthedocs.com](https://apgap.readthedocs.io/).
